# Blackmoor

Site da Blackmoor, um whisky escocês (Est. 1876). Landing page com a linha de produtos, drinks e cadastro.

## Tecnologias

- Astro
- TypeScript
- Tailwind CSS

## Rodando localmente

```bash
npm install
npm run dev
```

O site sobe em http://localhost:4321.

## Build

```bash
npm run build      # gera o site em dist/
npm run preview    # pré-visualiza o build
```

## Páginas

- `/` — home (hero + vídeo + linha de produtos)
- `/familia` — a família de rótulos (10, 12, XV, 18, XXI)
- `/drinks` — receitas de drinks
- `/sobre` — sobre a marca
- `/historias` — histórias
- `/cadastro` — cadastro
